# btop 系统监控

## 简介

btop 是现代化的系统监控工具，提供图形化的 CPU、内存、磁盘、网络监控界面。

## 安装

```bash
sudo apt install btop
```

---

## 启动

```bash
btop
```

---

## 快捷键

| 键 | 功能 |
|----|------|
| `q` | 退出 |
| `Esc` | 退出 |
| `m` | 切换视图模式 |
| `f` | 过滤进程 |
| `k` | 终止进程 |
| `Tab` | 切换面板 |
| `↑↓` | 选择进程 |
| `Enter` | 进程详情 |
| `?` | 显示帮助 |
| `1-4` | 切换面板 |

---

## 视图模式

按 `m` 切换：

1. **Full** - 显示所有信息
2. **Medium** - 中等信息量
3. **Compact** - 紧凑模式
4. **Minimal** - 最小模式

---

## 功能说明

### CPU 监控

- 实时使用率图表
- 各核心使用情况
- 温度显示
- 频率显示

### 内存监控

- 总内存使用情况
- Swap 使用情况
- 缓存信息

### 磁盘监控

- 磁盘使用率
- 读写速度
- 多磁盘支持

### 网络监控

- 上传/下载速度
- 网络连接数
- 网卡信息

### 进程管理

- 进程列表
- CPU/内存占用排序
- 进程搜索
- 终止进程

## 配置文件

配置文件位置：`~/.config/btop/btop.conf`

常用配置：

```bash
# 主题
color_theme = "TTY"

# 更新频率（毫秒）
update_ms = 1000

# 显示温度
show_temp = true

# 显示电池
show_battery = true
```

## 与其他工具对比

| 工具 | 特点 |
|------|------|
| top | 基础，界面简陋 |
| htop | 彩色，功能丰富 |
| btop | 图形化，美观，信息全面 |
| glances | 更全面，支持远程 |
