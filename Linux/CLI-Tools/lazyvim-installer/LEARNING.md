# LazyVim 新手练习指南

## 一、窗口切换练习

### 练习 1: 文件树切换
1. 按 `<Space>e` 打开文件树
2. 按 `<C-l>` 切换回编辑区
3. 按 `<C-h>` 切换到文件树
4. 按 `q` 关闭文件树

### 练习 2: 多窗口移动
1. 按 `<Space>e` 打开文件树
2. 按 `<C-h>` 到文件树
3. 按 `<C-l>` 到编辑区
4. 按 `<C-j>` 到下方窗口（如果有）
5. 按 `<C-k>` 到上方窗口（如果有）

---

## 二、文件操作练习

### 练习 3: 创建和打开文件
1. 按 `<Space>e` 打开文件树
2. 按 `a` 新建文件
3. 输入文件名（如 `test.lua`）
4. 按 `<Enter>` 打开文件

### 练习 4: 搜索文件
1. 按 `<Space>ff`
2. 输入文件名的一部分
3. 用 `j/k` 选择
4. 按 `<Enter>` 打开

### 练习 5: 搜索内容
1. 按 `<Space>fg`
2. 输入要搜索的文字
3. 用 `j/k` 选择结果
4. 按 `<Enter>` 跳转

---

## 三、编辑练习

### 练习 6: 基本编辑
1. 按 `i` 进入插入模式
2. 输入一些文字
3. 按 `Esc` 返回普通模式
4. 按 `u` 撤销
5. 按 `Ctrl-r` 重做

### 练习 7: 复制粘贴
1. 按 `v` 进入可视模式
2. 用 `j/k` 选择行
3. 按 `y` 复制
4. 移动光标
5. 按 `p` 粘贴

### 练习 8: 删除
1. 按 `dd` 删除整行
2. 按 `p` 粘贴回来

---

## 四、快速移动练习

### 练习 9: 行内移动
```
测试行：hello world this is a test line
```
1. 按 `w` 跳到下一个单词
2. 按 `b` 跳到上一个单词
3. 按 `0` 跳到行首
4. 按 `$` 跳到行尾

### 练习 10: 文件内移动
1. 按 `gg` 跳到文件开头
2. 按 `G` 跳到文件结尾
3. 按 `Ctrl-d` 向下翻半页
4. 按 `Ctrl-u` 向上翻半页

---

## 五、保存退出

| 操作 | 命令 |
|------|------|
| 保存 | `<Space>ww` 或 `:w` |
| 退出 | `:q` |
| 保存并退出 | `:wq` 或 `ZZ` |
| 强制退出不保存 | `:q!` |

---

## 六、常用快捷键速查

### 文件树 (neo-tree)
| 键 | 功能 |
|----|------|
| `j/k` | 上下移动 |
| `h` | 折叠/返回上级 |
| `l` | 展开/打开文件 |
| `a` | 新建 |
| `d` | 删除 |
| `r` | 重命名 |
| `?` | 显示帮助 |

### 窗口切换
| 键 | 功能 |
|----|------|
| `<C-h>` | 左窗口 |
| `<C-l>` | 右窗口 |
| `<C-j>` | 下窗口 |
| `<C-k>` | 上窗口 |

### Leader 键操作 (<Space>)
| 键 | 功能 |
|----|------|
| `ff` | 查找文件 |
| `fg` | 搜索内容 |
| `e` | 文件树 |
| `ww` | 保存 |
| `bd` | 关闭缓冲区 |
