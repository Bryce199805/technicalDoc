# Markdown 特殊字符与转义

## 1. 转义字符

### 1.1 可转义的 Markdown 字符

使用反斜杠 `\` 进行转义：

| 字符 | 名称 | 转义语法 | 渲染效果 |
|:----:|------|----------|:--------:|
| `\` | 反斜杠 | `\\` | \\ |
| `` ` `` | 反引号 | `` \` `` | \` |
| `*` | 星号 | `\*` | \* |
| `_` | 下划线 | `\_` | \_ |
| `{}` | 花括号 | `\{ \}` | \{ \} |
| `[]` | 方括号 | `\[ \]` | \[ \] |
| `()` | 圆括号 | `\( \)` | \( \) |
| `#` | 井号 | `\#` | \# |
| `+` | 加号 | `\+` | \+ |
| `-` | 连字符 | `\-` | \- |
| `.` | 句点 | `\.` | \. |
| `!` | 感叹号 | `\!` | \! |
| `|` | 竖线 | `\|` | \| |

### 1.2 转义示例

```markdown
\*这不是斜体\*
\# 这不是标题
\[这不是链接\](url)
```

**渲染效果：**

\*这不是斜体\*
\# 这不是标题
\[这不是链接\](url)

## 3. 空格字符

### 3.1 各种空格类型

| 类型 | 实体 | 效果 | 说明 |
|------|------|------|------|
| 不换行空格 | `&nbsp;` | a&nbsp;b | 宽度为标准空格，不会换行 |
| 半角空格 | `&ensp;` | a&ensp;b | 宽度为半个汉字 |
| 全角空格 | `&emsp;` | a&emsp;b | 宽度为一个汉字 |
| 窄空格 | `&thinsp;` | a&thinsp;b | 比标准空格窄 |

### 3.2 零宽字符

| 类型 | 实体 | 用途 |
|------|------|------|
| 零宽空格 | `&#8203;` | 不可见的分隔符 |
| 零宽非连字 | `&#8204;` | 防止连字 |
| 零宽连字 | `&#8205;` | 强制连字 |
| 左至右标记 | `&lrm;` | 文字方向控制 |
| 右至左标记 | `&rlm;` | 文字方向控制 |

## 9. Emoji 表情

### 9.1 在 Markdown 中使用 Emoji

## 10. 在代码块中显示特殊字符

代码块中的特殊字符无需转义：

~~~markdown
```
特殊字符：* _ # < > &
HTML实体：&lt; &gt; &amp;
```
~~~
