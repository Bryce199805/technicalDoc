## 快速索引

### 基础元素速查

| 元素 | 语法 | 效果 |
|------|------|------|
| 一级标题 | `# 标题` | 最大的标题 |
| 二级标题 | `## 标题` | 次级标题 |
| 加粗 | `**文字**` | **文字** |
| 斜体 | `*文字*` | *文字* |
| 删除线 | `~~文字~~` | ~~文字~~ |
| 行内代码 | `` `代码` `` | `代码` |
| 链接 | `[文字](URL)` | 可点击链接 |
| 图片 | `![描述](URL)` | 显示图片 |
| 无序列表 | `- 项目` | 列表项 |
| 有序列表 | `1. 项目` | 编号列表 |

### 数学公式快速参考

| 类型 | 语法示例 | 渲染效果 |
|------|----------|----------|
| 行内公式 | `$E=mc^2$` | 行内显示 |
| 块级公式 | `$$E=mc^2$$` | 独立居中 |
| 分数 | `\frac{a}{b}` | a/b |
| 根号 | `\sqrt{x}` | √x |
| 求和 | `\sum_{i=1}^{n}` | ∑ |
| 积分 | `\int_{a}^{b}` | ∫ |
| 矩阵 | `\begin{matrix}...\end{matrix}` | 矩阵形式 |

> 详细公式语法请参考 [04-数学公式](./04-数学公式.md)，包含 200+ LaTeX 数学符号表格。
