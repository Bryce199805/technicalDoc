# Markdown 表格语法

## 1. 基本表格

### 语法结构

```markdown
| 列1标题 | 列2标题 | 列3标题 |
|--------|--------|--------|
| 内容1  | 内容2  | 内容3  |
| 内容4  | 内容5  | 内容6  |
```

### 渲染效果

| 列1标题 | 列2标题 | 列3标题 |
|--------|--------|--------|
| 内容1  | 内容2  | 内容3  |
| 内容4  | 内容5  | 内容6  |

---

## 2. 表格对齐

使用冒号 `:` 控制列对齐方式。

### 对齐语法表

| 对齐方式 | 语法 | 示例 |
|----------|------|------|
| 左对齐 | `:---` | `|:---|` |
| 右对齐 | `---:` | `|---:|` |
| 居中对齐 | `:---:` | `:---:` |
| 默认 | `---` | `|---|` |

### 示例

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   |   内容   |   内容 |
| 长内容 |  长内容  | 长内容 |
| 123    |   456    |    789 |
```

**渲染效果：**

| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   |   内容   |   内容 |
| 长内容 |  长内容  | 长内容 |
| 123    |   456    |    789 |

## 3. 表格样式美化

### 3.1 可选的边框

表格两端的竖线 `|` 是可选的：

```markdown
列1 | 列2 | 列3
----|-----|----
A   | B   | C
```

等价于：

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
```

### 3.2 分隔线长度

分隔线长度不影响渲染，可以任意长度：

```markdown
| 列1 | 列2 |
|---|---|
| A | B |
```

## 4. 表格内容格式化

### 4.1 表格中的文本格式

| 元素 | 语法 |
|------|------|
| 加粗 | `**文字**` |
| 斜体 | `*文字*` |
| 链接 | `[文字](URL)` |
| 代码 | `` `代码` `` |
| 删除线 | `~~文字~~` |

### 示例

```markdown
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| **复制** | `Ctrl+C` | *复制选中内容* |
| **粘贴** | `Ctrl+V` | [查看帮助](#) |
| **删除** | `Delete` | ~~已废弃~~ |
```

**渲染效果：**

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| **复制** | `Ctrl+C` | *复制选中内容* |
| **粘贴** | `Ctrl+V` | [查看帮助](#) |
| **删除** | `Delete` | ~~已废弃~~ |

### 4.2 表格中的特殊字符

| 字符 | 转义方式 |
|------|----------|
| `\|` | 使用 `\|` 或 `&#124;` |
| 换行 | 使用 `<br>` |

```markdown
| 字段 | 值 |
|------|-----|
| 路径 | `C:\Users\name` |
| 描述 | 第一行<br>第二行 |
```

## 5. 复杂表格

### 5.1 表格中的列表

使用 HTML `<br>` 实现多行内容：

```markdown
| 项目 | 内容 |
|------|------|
| 项目A | • 子项1<br>• 子项2<br>• 子项3 |
```

### 5.2 表格中的代码块

标准 Markdown 表格不支持代码块，可以使用 HTML 表格：

```html
<table>
<tr>
<td>列1</td>
<td>

```python
def hello():
    print("Hello")
```

</td>
</tr>
</table>
```

## 6. HTML 表格

当 Markdown 表格不够用时，可以使用 HTML 创建复杂表格。

### 6.1 合并单元格

```html
<table>
  <tr>
    <th>姓名</th>
    <th colspan="2">联系方式</th>
  </tr>
  <tr>
    <td>张三</td>
    <td>电话</td>
    <td>邮箱</td>
  </tr>
  <tr>
    <td rowspan="2">李四</td>
    <td>138xxxx</td>
    <td>a@b.com</td>
  </tr>
  <tr>
    <td>139xxxx</td>
    <td>c@d.com</td>
  </tr>
</table>
```

### 6.2 HTML 表格属性

| 属性 | 值 | 说明 |
|------|-----|------|
| `border` | 数字 | 边框宽度 |
| `cellpadding` | 数字 | 单元格内边距 |
| `cellspacing` | 数字 | 单元格间距 |
| `width` | 数值/% | 表格宽度 |
| `colspan` | 数字 | 横向合并列数 |
| `rowspan` | 数字 | 纵向合并行数 |

## 7. 表格样式 (CSS)

### 7.1 内联样式

```html
<table style="width: 100%; border-collapse: collapse;">
  <tr style="background-color: #f0f0f0;">
    <th style="padding: 10px; border: 1px solid #ddd;">标题1</th>
    <th style="padding: 10px; border: 1px solid #ddd;">标题2</th>
  </tr>
  <tr>
    <td style="padding: 10px; border: 1px solid #ddd;">内容1</td>
    <td style="padding: 10px; border: 1px solid #ddd;">内容2</td>
  </tr>
</table>
```

### 7.2 常用表格样式

```css
/* 斑马纹表格 */
tr:nth-child(even) {
  background-color: #f9f9f9;
}

/* 鼠标悬停效果 */
tr:hover {
  background-color: #f5f5f5;
}

/* 表头样式 */
th {
  background-color: #4CAF50;
  color: white;
}
```

## 9. 表格最佳实践

### 9.1 何时使用表格

| 适用场景 | 不适用场景 |
|----------|------------|
| 对比数据 | 长段落文本 |
| 规格参数 | 纯文字描述 |
| 时间线 | 简单列表 |
| 短文本映射 | 复杂嵌套结构 |

### 9.2 表格设计原则

| 原则 | 说明 |
|------|------|
| 简洁 | 避免过多列（建议 ≤ 5 列） |
| 对齐 | 数字右对齐，文本左对齐 |
| 一致 | 同类数据格式统一 |
| 易读 | 使用适当的空格和对齐 |

## 常见问题

### Q: 如何在表格中换行？

A: 使用 `<br>` 标签：
```markdown
| 列1 | 列2 |
|-----|-----|
| 第一行<br>第二行 | 内容 |
```

### Q: 如何在表格中显示竖线？

A: 使用 HTML 实体 `&#124;` 或转义 `\|`

### Q: 为什么我的表格没有渲染？

A: 检查：
- 分隔行是否正确（`|---|` 格式）
- 是否有空行分隔
- 表格前后是否有必要的空行
