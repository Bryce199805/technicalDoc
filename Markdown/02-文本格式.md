# Markdown 文本格式

## 1. 强调语法速查表

| 效果 | 语法 | 快捷键 | 示例 |
|------|------|--------|------|
| **加粗** | `**文字**` 或 `__文字__` | Ctrl+B | `**粗体**` |
| *斜体* | `*文字*` 或 `_文字_` | Ctrl+I | `*斜体*` |
| ***粗斜体*** | `***文字***` 或 `___文字___` | - | `***粗斜体***` |
| ~~删除线~~ | `~~文字~~` | - | `~~删除~~` |
| ==高亮== | `==文字==` | - | `==高亮==` |

### 示例

```markdown
**这是加粗文本**
__这也是加粗文本__

*这是斜体文本*
_这也是斜体文本_

***这是粗斜体文本***
___这也是粗斜体文本___

~~这是删除线文本~~

==这是高亮文本==（需要扩展支持）
```

## 2. 文本格式详细说明

### 2.1 加粗 (Bold)

| 语法 | 说明 |
|------|------|
| `**文字**` | 推荐，使用星号 |
| `__文字__` | 使用下划线 |

```markdown
**重要内容**
__另一种加粗方式__
```

**注意：** 两种语法效果相同，但推荐使用星号，因为下划线可能与链接语法冲突。

### 2.2 斜体 (Italic)

| 语法 | 说明 |
|------|------|
| `*文字*` | 推荐，使用星号 |
| `_文字_` | 使用下划线 |

```markdown
*强调内容*
_另一种斜体方式_
```

### 2.3 粗斜体 (Bold + Italic)

| 语法 | 说明 |
|------|------|
| `***文字***` | 三个星号（推荐） |
| `___文字___` | 三个下划线 |
| `**_文字_**` | 混合使用 |
| `__*文字*__` | 混合使用 |

### 2.4 删除线 (Strikethrough)

```markdown
~~已删除的内容~~
```

**渲染效果：** ~~已删除的内容~~

> 删除线是 GFM（GitHub Flavored Markdown）扩展语法。

### 2.5 高亮 (Highlight)

```markdown
==高亮文本==
```

**渲染效果：** ==高亮文本==

> 高亮语法需要特定扩展支持（如 Obsidian），标准 Markdown 不支持。

## 3. 下标与上标

### 3.1 HTML 方式（通用）

| 效果 | 语法 | 示例 |
|------|------|------|
| 上标 | `<sup>文字</sup>` | `x<sup>2</sup>` → x² |
| 下标 | `<sub>文字</sub>` | `H<sub>2</sub>O` → H₂O |

```markdown
E = mc<sup>2</sup>
H<sub>2</sub>O
log<sub>10</sub>100 = 2
```

### 3.2 扩展语法

部分 Markdown 解析器支持：

```markdown
H~2~O      # 下标
x^2^       # 上标
```

## 4. 脚注 (Footnotes)

### 基本语法

```markdown
这是一个脚注引用[^1]。

[^1]: 这是脚注的内容。

另一个脚注[^note]。

[^note]: 可以使用任意标签。
```

### 完整示例

```markdown
Markdown 是一种轻量级标记语言[^gruber]，由 John Gruber 于 2004 年创建[^history]。

[^gruber]: John Gruber 是 Markdown 的创始人。
[^history]: Markdown 的设计目标是让文档更易读易写。
```

**渲染效果：**

Markdown 是一种轻量级标记语言[^gruber]，由 John Gruber 于 2004 年创建。

[^gruber]: John Gruber 是 Markdown 的创始人。

### 脚注语法表

| 语法 | 说明 |
|------|------|
| `[^label]` | 引用脚注 |
| `[^label]: 内容` | 定义脚注 |

## 5. 行内代码 (Inline Code)

### 基本语法

使用反引号包裹代码：

```markdown
使用 `console.log()` 输出调试信息。
```

**渲染效果：** 使用 `console.log()` 输出调试信息。

### 包含反引号的代码

如果代码本身包含反引号，使用双反引号包裹：

```markdown
``代码中包含 ` 反引号``
```

### 行内代码中的特殊字符

在行内代码中，特殊字符不需要转义：

```markdown
`*这不是斜体*`
`**这不是加粗**`
`[这不是链接](url)`
```

## 6. 键盘按键

使用 HTML `<kbd>` 标签表示键盘按键：

```markdown
按 <kbd>Ctrl</kbd> + <kbd>C</kbd> 复制。
按 <kbd>Ctrl</kbd> + <kbd>V</kbd> 粘贴。
```

**渲染效果：** 按 <kbd>Ctrl</kbd> + <kbd>C</kbd> 复制。

### 常用快捷键表格

| 操作 | Windows/Linux | macOS |
|------|---------------|-------|
| 复制 | <kbd>Ctrl</kbd>+<kbd>C</kbd> | <kbd>⌘</kbd>+<kbd>C</kbd> |
| 粘贴 | <kbd>Ctrl</kbd>+<kbd>V</kbd> | <kbd>⌘</kbd>+<kbd>V</kbd> |
| 撤销 | <kbd>Ctrl</kbd>+<kbd>Z</kbd> | <kbd>⌘</kbd>+<kbd>Z</kbd> |
| 全选 | <kbd>Ctrl</kbd>+<kbd>A</kbd> | <kbd>⌘</kbd>+<kbd>A</kbd> |
| 保存 | <kbd>Ctrl</kbd>+<kbd>S</kbd> | <kbd>⌘</kbd>+<kbd>S</kbd> |

## 7. 缩写 (Abbreviations)

部分解析器支持缩写定义：

```markdown
HTML 是一种标记语言。

*[HTML]: HyperText Markup Language
```

鼠标悬停在 HTML 上时会显示完整定义。

---

## 8. 文本对齐

Markdown 本身不支持文本对齐，需要使用 HTML：

```html
<div align="left">左对齐</div>
<div align="center">居中对齐</div>
<div align="right">右对齐</div>
```

或者使用 CSS：

```html
<p style="text-align: center;">居中段落</p>
```

## 9. 文本颜色

使用 HTML 设置文本颜色：

```html
<span style="color: red;">红色文本</span>
<span style="color: #FF5733;">自定义颜色</span>
<span style="color: rgb(255, 87, 51);">RGB颜色</span>
<span style="background-color: yellow;">高亮背景</span>
```

### 颜色值表示方式

| 方式 | 语法 | 示例 |
|------|------|------|
| 颜色名 | `color: red` | red, blue, green |
| 十六进制 | `color: #FF5733` | #RRGGBB |
| RGB | `color: rgb(255,87,51)` | rgb(r,g,b) |
| RGBA | `color: rgba(255,87,51,0.5)` | rgba(r,g,b,a) |

## 10. 特殊文本效果

### 10.1 注释

HTML 注释在 Markdown 中有效，渲染时不会显示：

```markdown
<!-- 这是注释，不会显示 -->

可见内容
<!-- 另一个注释 -->
```

### 10.2 空格

| 类型 | 代码 | 效果 |
|------|------|------|
| 不换行空格 | `&nbsp;` | 不断开的空格 |
| 半角空格 | `&ensp;` | 宽度约半个汉字 |
| 全角空格 | `&emsp;` | 宽度约一个汉字 |

```markdown
前&nbsp;后    # 不换行空格
前&ensp;后    # 半角空格
前&emsp;后    # 全角空格
```

### 10.3 上下划线

```html
<span style="text-decoration: overline;">上划线</span>
<span style="text-decoration: underline;">下划线</span>
<span style="text-decoration: line-through;">删除线</span>
<span style="text-decoration: underline overline;">上下划线</span>
```

## 11. 文本格式组合

### 组合规则

| 组合 | 语法 | 效果 |
|------|------|------|
| 加粗斜体 | `***文字***` | ***粗斜体*** |
| 加粗代码 | 不支持 | - |
| 斜体链接 | `*[文字](URL)*` | *链接* |
| 加粗链接 | `**[文字](URL)**` | **链接** |
| 删除加粗 | `~~**文字**~~` | ~~**删除粗体**~~ |

### 链接与强调组合

```markdown
[普通链接](https://example.com)
*[斜体链接](https://example.com)*
**[加粗链接](https://example.com)**
***[粗斜体链接](https://example.com)***
~~[删除链接](https://example.com)~~
```

## 常见问题

### Q: 为什么我的 `*` 没有变成斜体？

A: 检查以下几点：
- 是否在 `*` 前后有正确的空格
- 是否与文字紧密相连
- 是否在代码块或行内代码中

### Q: 如何显示星号本身而不是斜体？

A: 使用反斜杠转义：`\*` 或使用行内代码：`` `*` ``

### Q: 下划线和星号有什么区别？

A: 效果相同，但推荐使用星号，原因：
- 下划线可能与链接语法混淆
- 星号更常见且可读性更好
