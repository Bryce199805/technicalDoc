# Markdown 代码块语法

## 1. 基本语法

### 1.1 行内代码

使用反引号包裹：

```markdown
使用 `print()` 函数输出。
```

**渲染效果：** 使用 `print()` 函数输出。

### 1.2 代码块

使用三个反引号包裹：

~~~markdown
```
普通代码块
```
~~~

### 1.3 语法高亮

在开头的反引号后指定语言：

~~~markdown
```python
def hello():
    print("Hello, World!")
```
~~~

**渲染效果：**

```python
def hello():
    print("Hello, World!")
```

## 3. 代码块高级功能

### 3.1 显示文件名

部分平台支持：

~~~markdown
```python title="hello.py"
def hello():
    print("Hello, World!")
```
~~~

或：

~~~markdown
```python {title="hello.py"}
def hello():
    print("Hello, World!")
```
~~~

### 3.2 行号

~~~markdown
```python linenums="1"
def hello():
    print("Hello, World!")
```
~~~

### 3.3 高亮指定行

~~~markdown
```python hl_lines="2 3"
def hello():
    print("Hello, World!")  # 高亮
    return True              # 高亮
```
~~~

### 3.4 代码折叠

使用 HTML 实现折叠代码块：

```html
<details>
<summary>点击查看代码</summary>

```python
def hello():
    print("Hello, World!")
```

</details>
```

## 常见问题

### Q: 为什么我的代码没有高亮？

A: 检查：
- 是否指定了正确的语言标识符
- 该语言是否被渲染器支持
- 代码块是否正确闭合

### Q: 如何在代码块中显示特殊字符？

A: 代码块中的特殊字符通常不需要转义，直接显示即可。

### Q: 如何复制代码块内容？

A: 现代渲染器通常提供复制按钮，或可手动选择复制。
